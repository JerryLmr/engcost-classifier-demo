# 灰区补充材料清单 V1

用途：
服务于 gray_case_review_flow。
弱灰区对象不是直接人工复核，而是先要求补充“消解灰区”的证据。

## 一、通用灰区补充材料

| 材料 | 用途 |
|---|---|
| 维修部位说明 | 判断是否属于共用部位/共用设施 |
| 维修前现场照片 | 判断损坏范围和性质 |
| 损坏情况说明 | 判断是日常小修还是明显维修 |
| 责任边界说明 | 判断是否属于物业服务合同维保范围 |

## 二、楼道窗户玻璃维修专用

| 材料 | 用途 |
|---|---|
| 维修部位平面说明/位置说明 | 判断是否为公共窗户、楼道共用部位 |
| 维修前现场照片 | 判断损坏范围 |
| 损坏数量及范围说明 | 判断单点破损还是系统性损坏 |
| 物业服务合同中维保责任条款（如有） | 判断是否属于物业维保范围 |
| 维修方案或情况说明 | 判断是简单更换还是整体修复 |
| 业主/业委会说明（如有） | 判断是否按共用部分维修申报 |

### 补证后的处理逻辑
- 若仍无法明确是否属于物业维保 -> manual_review
- 若明确属于共用部位，且不属于物业维保 -> 转 normal_flow
- 若明确属于物业维保 -> non_compliant

## 三、弱电修复专用

| 材料 | 用途 |
|---|---|
| 维修对象说明 | 判断是修复原功能还是新增/优化 |
| 施工前后对比说明 | 防止“维修”名义下做升级 |
| 原设备损坏证明/故障说明 | 判断维修必要性 |
| 是否涉及位置调整/扩容说明 | 一旦涉及，转 gray_case 或 new_construction |

## 四、门窗/附属设施修复专用

| 材料 | 用途 |
|---|---|
| 所属部位说明 | 是否共用部位 |
| 维修范围说明 | 个别小修还是系统性损坏 |
| 是否纳入物业日常维修清单说明 | 判断维保边界 |
| 现场照片 | 验证损坏性质 |
